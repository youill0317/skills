# Plan Template

Use this template to produce a decision-complete implementation plan.

## 1. Goal and Success Criteria

- State the objective in one sentence.
- Define measurable completion conditions.
- Clarify intended audience or consumers.

## 2. Scope

- In scope
- Out of scope
- Deferred items

## 3. Constraints and Dependencies

- Technical constraints
- Environment constraints
- External dependencies

## 4. Interfaces and Behavior Changes

- API/type/schema changes
- Input/output changes
- Backward compatibility notes

## 5. Implementation Steps

- Ordered steps
- Owner or component per step
- Done condition for each step

## 6. Tests and Validation

- Unit scenarios
- Integration scenarios
- Edge and failure scenarios
- Acceptance criteria

## 7. Risks and Mitigations

- Key risks
- Probability/impact notes
- Concrete mitigation actions

## 8. Assumptions and Defaults

- Explicit assumptions
- Default choices used when unspecified
