# Search Playbook

Use this playbook to execute exhaustive bilingual research consistently.

## Query Expansion Checklist

For each user question, create variations in both the user language and English:

1. Core question phrasing
2. Synonym phrasing
3. Entity + claim phrasing
4. Date/recency phrasing
5. Comparison phrasing
6. Counterclaim phrasing

## Search Rounds

Run these rounds in order and repeat as needed:

1. Broad discovery:
   - collect candidate sources and key entities
2. Focused fact extraction:
   - validate names, dates, numbers, and definitions
3. Independent verification:
   - confirm claims through separate sources
4. Conflict resolution:
   - isolate disagreement points and re-query them
5. Recency validation:
   - confirm latest changes and effective dates
6. Gap fill:
   - search missing context, edge cases, and alternative views

## Coverage Targets

- Simple query: at least 2 rounds and independent verification.
- Medium query: at least 4 rounds with comparative checks.
- Complex query: full cycle, aggressive expansion, target 100+ tool uses when feasible.

## Stop Conditions

Stop only when all are true:

1. Core claims are independently verified.
2. Contradictions are resolved or explicitly documented.
3. Recency checks are complete for time-sensitive claims.
4. Answer can be cited with stable evidence IDs.
