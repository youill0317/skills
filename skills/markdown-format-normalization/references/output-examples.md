# Output Examples

Load this reference when the target Markdown shape needs a concrete example.

## Example: Clean Heading and Spacing

```markdown
# Title

## Section

Paragraph text.

### Subsection

More text.
```

## Example: Normalized List

```markdown
- First item
- Second item
    - Nested item
```

## Example: Normalized Table

```markdown
| Item | Detail |
|:---|:---|
| A | Description |
| B | More detail |
```

## Example: Basic Frontmatter

```yaml
---
tags: [notes, cleaned]
---
```
