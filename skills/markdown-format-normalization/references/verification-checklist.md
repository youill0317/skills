# Verification Checklist

Load this reference before finalizing normalized Markdown.

## Structural Checks

1. Markdown is syntactically coherent.
2. Heading depth is internally consistent.
3. Lists render as intended.
4. Tables have stable columns or were intentionally rewritten as lists.
5. Callouts, quotes, code, math, and footnotes remain intact.

## Preservation Checks

1. Meaning and content order were preserved.
2. Links and images were not changed.
3. No content was dropped during cleanup.
4. Frontmatter remains valid if present.

## Final Rule

If a normalization choice would require guessing the intended content, do not guess. Preserve the source text and apply only safe formatting changes.
